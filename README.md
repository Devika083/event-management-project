# Event Management Project

A simple Event Management website built using HTML.

## 📌 About
This project is a basic web page designed to manage and display event details in a clean and simple layout.

## 🚀 Features
- Simple and clean UI
- Event details display
- Image support (img folder)
- Built with pure HTML

## 🛠️ Tech Stack
- HTML
